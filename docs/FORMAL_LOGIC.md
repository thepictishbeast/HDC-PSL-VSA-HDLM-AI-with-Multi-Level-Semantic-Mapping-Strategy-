# Formal Logic Ingestor (src/data_ingestion/formal_logic.rs)
**Status:** Alpha-Active
**Protocol:** First-Principles Vectorization

## Overview
This module bridges the gap between formal mathematical proofs and holographic associative memory. It ingests `.lean` files from the Lean Mathlib ecosystem and binds theorem-proof pairs into the 10,000-D VSA substrate.

## Functional Logic
- **Theorem Anchoring:** Extracts the `theorem` definition as the key hypervector.
- **Proof Binding:** Extracts the `proof` block and binds it to the theorem using XOR-equivalent element-wise multiplication.
- **Retrieval:** Allows the BigBrain to "recall" a formal proof by probing the memory bus with a theorem vector.

## Unit Testing
The module includes `test_lean_ingestion_logic` and `test_vsa_logic_retrieval` to prove that the mathematical integrity of the logic grains is maintained during the binding process.
