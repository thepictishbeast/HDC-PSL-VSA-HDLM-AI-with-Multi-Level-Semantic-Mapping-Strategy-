  • Cannot distinguish real reasoning from confabulation
  • Vulnerable to every known prompt injection
  • Run on cloud servers (subpoenable, logged, monitored)
  • Detect AI attacks using pre-AI spam filters
  • Output explanations that are generated after the answer

  • Confidence is asymptotically bounded below 1.0
